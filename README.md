# Words_Recite_Process
单词背诵软件（提供Windows版、Linux版）
一、账户注册：输入账号、姓名、密码创建账户，账号与姓名不可与已存在的账户重复，注册后信息存储在system/user.dat中，并根据账号名在userdata中创建对应目录，目录内存储该账户的错题信息。

二、账户登录：输入账号、密码，系统读取system/user.dat中数据判断账号、密码是否正确，是则成功登录。
	1、每日打卡：每日可打卡一次并记录打卡总数，同一天内不可多次打卡。
	2、单词学习：按10个/页展示单词本，可切换上一页、下一页、指定页或通过序号/英文搜索指定单词。
	3、题目练习：可进行选择题、拼写题练习，输入题目数(1-100)，系统自动根据单词本内容生成练习题，自动记录错题与分数。
	4、成绩记录：可查看最近20次成绩、清空最近成绩。
	5、错题复习：按5题/页查看选择题/拼写题错题，可切换上一页、下一页、指定页、通过序号搜索指定错题、通过序号指定错题、删除所有错题。
	
三、高级设置：首次进入时要求设置系统高级密码并存储在system/system_passward.bat中，之后每次进入都需要先输入密码。
	1、账户管理：系统读取system/user.dat中的信息，可通过序号/账号/姓名修改/删除指定账户（可删除所有账户）、按10个/页展示账户表，可切换上一页、下一页、指定页或通过序号/账号/姓名搜索指定账户。
	2、单词本管理：系统读取system/wordsbook.dat中的信息，可输入英文、中文、词性添加单词、通过序号/英文修改/删除指定单词（可删除所有单词）、按10个/页展示单词本，可切换上一页、下一页、指定页或通过序号/英文搜索指定单词。
	3、备份数据：可输入备份名称进行文件备份，备份名不可与已存在的备份名重复，备份名信息存储在backup/backuplog.dat中，并根据备份名在backup中创建对应目录，目录中存储当前system与userdata目录中所有文件。
	4、恢复数据：系统读取backup/backuplog.dat中的信息，按5条/页展示备份名，可切换上一页、下一页、指定页或通过备份名搜索指定备份后恢复账户/单词本/系统密码/所有数据，或通过备份名删除指定备份（可删除所有备份）。
	5、重置所有数据：将backup、system、userdata目录恢复至原始状态。
	6、更改系统密码：输入新密码后存储在system/system_passward.bat中。
	
四、测试单词本：Windows/Linux文件夹内提供对应版本的已存储500词的单词本wordsbook.dat用于测试，使用时替换掉主程序目录下的system/wordsbook.dat
