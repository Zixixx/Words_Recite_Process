# 单词背诵软件功能说明

**支持平台**：Windows版、Linux版

---

## 一、账户注册

- 输入**账号**、**姓名**、**密码**创建账户
- **账号**与**姓名**不可与已存在的账户重复
- 注册后信息存储在 `system/user.dat` 中
- 根据账号名在 `userdata` 中创建对应目录，目录内存储该账户的错题信息

---

## 二、账户登录

- 输入**账号**、**密码**
- 系统读取 `system/user.dat` 中数据判断账号、密码是否正确，是则成功登录

### 1. 每日打卡

- 每日可打卡一次并记录打卡总数
- 同一天内不可多次打卡

### 2. 单词学习

- 按 **10个/页** 展示单词本
- 可切换**上一页**、**下一页**、**指定页**
- 可通过**序号**或**英文**搜索指定单词

### 3. 题目练习

- 可进行**选择题**、**拼写题**练习
- 输入题目数（1-100）
- 系统自动根据单词本内容生成练习题
- 自动记录**错题**与**分数**

### 4. 成绩记录

- 可查看**最近20次成绩**
- 可**清空最近成绩**

### 5. 错题复习

- 按 **5题/页** 查看选择题/拼写题错题
- 可切换**上一页**、**下一页**、**指定页**
- 可通过**序号搜索**指定错题
- 可通过**序号指定**错题
- 可**删除所有错题**

---

## 三、高级设置

- **首次进入**时要求设置系统高级密码，并存储在 `system/system_password.bat` 中
- **之后每次进入**都需要先输入密码

### 1. 账户管理

- 系统读取 `system/user.dat` 中的信息
- 可通过**序号/账号/姓名**修改/删除指定账户（可删除所有账户）
- 按 **10个/页** 展示账户表
- 可切换**上一页**、**下一页**、**指定页**
- 可通过**序号/账号/姓名**搜索指定账户

### 2. 单词本管理

- 系统读取 `system/wordsbook.dat` 中的信息
- 可**输入英文、中文、词性**添加单词
- 可通过**序号/英文**修改/删除指定单词（可删除所有单词）
- 按 **10个/页** 展示单词本
- 可切换**上一页**、**下一页**、**指定页**
- 可通过**序号/英文**搜索指定单词

### 3. 备份数据

- 可输入**备份名称**进行文件备份
- 备份名不可与已存在的备份名重复
- 备份名信息存储在 `backup/backuplog.dat` 中
- 根据备份名在 `backup` 中创建对应目录，目录中存储当前 `system` 与 `userdata` 目录中所有文件

### 4. 恢复数据

- 系统读取 `backup/backuplog.dat` 中的信息
- 按 **5条/页** 展示备份名
- 可切换**上一页**、**下一页**、**指定页**
- 可通过**备份名搜索**指定备份后恢复**账户/单词本/系统密码/所有数据**
- 可通过**备份名删除**指定备份（可删除所有备份）

### 5. 重置所有数据

- 将 `backup`、`system`、`userdata` 目录恢复至**原始状态**

### 6. 更改系统密码

- 输入新密码后存储在 `system/system_password.bat` 中

---

## 四、测试用单词本

- 提供已存储 **500词** 的单词本 `wordsbook.dat` 用于测试
- 使用时替换掉主程序目录下的 `system/wordsbook.dat`
